# STM32F103C8T6 PWM 输入捕获测频 — 开发流程

## 1. 项目概述

- **MCU**: STM32F103C8T6（Cortex-M3, 64KB Flash, 20KB SRAM, 72MHz）
- **外设**: TIM3 CH1 输入捕获 + TIM2 CH1 PWM 输出 + OLED 显示
- **通信方式**: 输入捕获（PA6, Reset 模式）+ 软件 I2C（OLED）
- **构建工具**: Keil MDK uVision5 / EIDE (VSCode 插件 `cl.eide`)
- **调试器**: ST-Link (SWD)
- **代码仓库**: [GitHub: rjsjing/stm32-pwm-input-capture](https://github.com/rjsjing/stm32-pwm-input-capture)

---

## 2. 技术栈

- **MCU**: STM32F103C8T6（Cortex-M3）
- **库**: ST 标准外设库 v3.5.0
- **构建工具**: Keil MDK uVision5 + EIDE (VSCode 插件 `cl.eide`)
- **调试器**: ST-Link (SWD)

---

## 3. 硬件连接

| STM32F103C8T6 | 连接目标 |
|---------------|----------|
| PA0           | PWM 信号输出（TIM2_CH1） |
| PA6           | PWM 信号输入（TIM3_CH1, 捕获） |
| PB5           | OLED SCL |
| PB6           | OLED SDA |
| 3.3V          | VCC     |
| GND           | GND     |

> PA0 和 PA6 需飞线连接（PWM 输出 → 捕获输入）

## 5. 开发步骤

### 5.1 依赖

- Keil MDK v5 + STM32F10x 设备包
- VSCode + EIDE 插件 (`cl.eide`)
- 编译宏: `USE_STDPERIPH_DRIVER`, `STM32F10X_MD`

### 5.2 编译

```
Keil:  打开 Project.uvprojx → F7 编译
EIDE:  Ctrl+Shift+P → "EIDE: Build"
```

### 5.3 烧录

```
Keil:  F8 烧录（ST-Link SWD）
CLI:   STM32_Programmer_CLI.exe -c port=SWD -w build/Target\ 1/Project.hex
```

### 5.4 功能验证

- PA0 输出 1kHz PWM，PA6 捕获测量
- OLED 实时显示测量频率 (Hz)
